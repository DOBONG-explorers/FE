## 🏞️ 프로젝트 개요
도봉마을탐험대는 **도봉구**의 숨은 명소와 매력을 발견하고,  
이를 **3D 모델링**, **AI 추천 시스템**, **Kakao Map 연동** 등을 통해 시각화하여  
지역 주민들이 쉽게 탐험할 수 있는 플랫폼을 제공하는 모바일 애플리케이션입니다.

## 🚀 기능 및 특징
- **3D 명소 탐험**: Blender로 제작한 도봉구 주요 명소의 3D 모델을 앱 내에서 실시간 렌더링하여 탐험.
- **AI 추천 서비스**: 사용자의 관심사에 맞춘 맞춤형 명소 추천 기능 제공.
- **지도 탐색**: Kakao Map SDK를 이용하여 도봉구의 주요 명소를 지도 상에 표시하고, 클릭 시 상세 정보 제공.
- **사용자 참여**: 리뷰 작성, 즐겨찾기 기능을 통해 사용자가 명소에 대해 의견을 남기고 소통할 수 있는 공간 제공.

## 🛠️ 기술 스택
- **언어**: Kotlin
- **플랫폼**: Android
- **디자인 도구**: Figma
- **지도 API**: Kakao Map SDK
- **3D 모델링**: Blender, Sceneform
- **네트워크 통신**: Retrofit2, OkHttp
- **이미지 로딩**: Glide
